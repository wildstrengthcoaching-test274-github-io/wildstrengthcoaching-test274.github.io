Diese Website wird über GitHub Pages zur Verfügung gestellt, was bedeutet, dass sie ein globales Vertriebsnetzwerk (CDN) verwendet, um Inhalte zuverlässig anzuzeigen. Wenn Sie die Website nutzen, können einige Ihrer nicht-personenbezogenen Daten, wie Ihre IP-Adresse und Informationen zur Seitennutzung, über dieses Netzwerk übertragen werden, um die ordnungsgemäße Funktion der Website sicherzustellen. Weitere Informationen finden Sie im Impressum .

Bitte bestätigen Sie, dass Sie damit einverstanden sind und dass wir ein Einwilligungs-Cookie verwenden, das für einen Zeitraum von einem Tag gespeichert wird. Dieses Cookie ermöglicht es uns, Ihre Zustimmung zur Verwendung nicht-personenbezogener Daten und zur ordnungsgemäßen Funktion der Website zu speichern.
